# Coinbase

## 기업 프로필

| 항목 | 내용 |
|------|------|
| 설립 | 2012년 (Brian Armstrong) |
| 본사 | 샌프란시스코 (원격근무) |
| 사업모델 | 크립토 거래소 + 기관 커스터디 + Base L2 + 스테이블코인 |
| 주요 지표 | 나스닥 상장, OCC 연방은행 인가 조건부 승인 (최초) |
| 투자 | a16z, Tiger Global 등 |
| 핵심 해자 | 규제 준수 선두, 연방은행 지위 추진, BTC 수익 펀드 온체인화 |

## 최근 시그널

### 코인베이스 연방은행 인가 조건부 승인
*2026-04-07 · TLDR*

**So What**: 암호화폐 거래소가 연방은행 지위를 얻으면 크립토의 제도권 편입이 본격화된다.

- OCC(통화감독청)의 국립은행 헌장 조건부 승인은 미국 크립토 거래소 최초 사례
- 조건 충족 후 연방 은행 감독 체계 적용, 기존 주(州)별 규제에서 연방 단위로 격상

### 코인베이스 BTC 수익 펀드 온체인화
*2026-03-25 · TLDR*

**So What**: 기관급 금융상품이 블록체인 위에서 규정 준수와 함께 실제 운용되는 사례가 본격화되고 있다.

- ERC-3643 표준으로 KYC/AML 컴플라이언스를 토큰 자체에 내장, 승인된 지갑만 보유·이전 가능
- 파트너 Apex Group은 작년 Tokeny 인수 후 T-REX Ledger로 2027년 6월까지 토큰화 펀드 1,000억 달러 목표
