> **테시스**: AI 에이전트의 자율성 확대는 필연적으로 새로운 보안 공격 면을 만들며, 기존 보안 패러다임으로는 대응이 불가능하다.

공급망 해킹, 하네스 유출, DeFi 키 탈취 — 에이전트 자율성이 보안 취약점으로 전환되는 패턴

### AI 에이전트 보안 프레임워크 ClawKeeper
*2026-04-04 · TLDR*

**So What**: 자율 에이전트 확산에 따른 보안 공백을 실시간으로 메우는 전문 프레임워크 등장.

- 명령어(instruction) 수준의 사전 보호 기능 내장
- 런타임 단계에서 에이전트 행동을 실시간 강제(enforcement) 제어

### AI 에이전트 시대의 공급망 공격
*2026-04-03 · a16z*

**So What**: AI가 코드를 쓰는 시대, 보안 검토 없는 자동 의존성 설치가 새로운 치명적 취약점이 됐다.

- Axios는 주간 1억 회 이상 다운로드되는 HTTP 라이브러리로, 소스코드 변경 없이 의존성 추가만으로 백도어가 심어졌다
- 일반 Next.js 프로젝트는 코드 한 줄 작성 전 이미 282개 패키지를 설치하며, GitHub JS 프로젝트의 이행적 의존성 중위값은 755개

### Claude Code 소스코드 유출
*2026-04-02 · TLDR*

**So What**: 경쟁사와 개발자들이 Anthropic의 핵심 구현 방식을 직접 들여다볼 수 있게 됐다.

- 가장 주목받은 발견: 컨텍스트 엔트로피 문제를 3계층 메모리 아키텍처로 해결한 방식
- 코드베이스는 수천 명의 개발자에 의해 미러링 및 분석됨

### 클로드 코드 하네스 설계 유출
*2026-04-01 · 안될공학*

**So What**: 모델 가중치가 아닌 '하네스 설계'가 AI 경쟁력의 핵심이 된 시대, 배포 실수 하나가 전략적 IP를 노출할 수 있다.

- 유출 규모: 소스맵 포함 실수로 약 50만 줄 내부 타입스크립트 코드 복원 가능해짐
- 앤트로픽은 2026년 3월 24일 공식 글에서 '하네스 디자인이 에이전트 성능의 핵심'이라고 직접 명시
